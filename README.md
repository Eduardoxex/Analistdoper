## 📌 Descripción General

**Analistdoper** (**CanMatch.io**) es el componente frontend de una plataforma inteligente enfocada en la automatización y optimización de procesos de reclutamiento y selección de personal mediante el uso de **Inteligencia Artificial**.

La plataforma permite analizar currículos (CVs) de manera automatizada, compararlos con vacantes específicas y generar métricas de compatibilidad que ayudan a las empresas a identificar candidatos ideales de forma rápida, objetiva y eficiente.

Este frontend fue desarrollado utilizando tecnologías modernas como **Angular**, ofreciendo una experiencia de usuario rápida, escalable y altamente interactiva.

---

# 🎯 Objetivos del Proyecto

- Automatizar procesos de análisis de talento humano.
- Reducir el tiempo de evaluación de candidatos.
- Mejorar la precisión en la selección de personal.
- Implementar herramientas inteligentes para reclutadores.
- Centralizar la gestión de candidatos en una sola plataforma.
- Crear una experiencia moderna y responsive para usuarios y empresas.
